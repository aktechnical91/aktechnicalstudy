Electrician: - The person who performs the works like manufacture, maintenance and repair of electrical appliances, is called electrician.
Power Factor (P.F.) :
The ratio of real power and virtual power in an AC circuit is called power factor.
[latex]P.F\;=\;\frac{V\ast I\;Cos\phi}{V\ast I} [/latex]
value of Power factor : The power factor has a maximum value of 1, and has no unit. The power factor is of the following type.
1. Unity power factor ie [latex]X_C=X_L[/latex]
2. Leading power factor ie [latex]X_C>X_L[/latex]
3. Lagging power factor ie [latex]X_C< X_L[/latex]
Note: - In a circuit containing inductors i.e. motors, electric fans etc., the power factor is reduced from the unit.

What is motherboard.?

A motherboard is the main printed circuit board (PCB) in a computer,that connects the different parts of a computer together.Motherboard is also known as a mainboard, planar board or logic board, system board,or MB.In addition to circuits, a motherboard contains a number of sockets and slots to connect the other components.

What is Battery ?

1. Primary Cell

Computer

Short Computer Notes

The CPU is also called the brain of a computer. The full name of the CPU is Central Processing Unit. And it is called central operating system in Hindi. CPU is a main part of a computer whose job is to process the input instructions coming into the computer. All the data of the computer is stored in the CPU itself.

central processing unit (CPU)

Italian physicist Federico Faggin invented the first commercial CPU. The Intel 4004 is a 4-bit central processing unit (CPU) released by Intel Corporation in 1971.

A hard disk is a storage device. It is a non-volatile memory hardware device that stores and retrieves data on a computer permanently. Hard disks are not the main memory of a computer. Programs and data remain in the disk until it is deleted by users.

Hard Drive

The first hard drive was invented by IBM in 1956.It held 5MB of data at a cost of $ 10,000 per megabyte.

A joystick is an input device. Joystick is used in flight simulator, computer gaming, CAD / CAM systems. It is similar to a stick. That is why it is called a joy stick. It has many buttons. The joystick and mouse function the same way.

Joystick

In 1972 the first video game joystick was invented by Ralph H. Bear inventor of TV video games.

A keyboard is a type of main input device that provides data and information to the computer in letters and numbers. When a key is pressed on the keyboard, the keyboard contacts the keyboard controller and the keyboard buffer.

keyboard

GrayGris Inc.

A mouse is an input device, originally named Pointing Device. It is mainly used in selecting, moving and opening and closing items on a computer screen. Through the mouse, the user instructs the computer. In 1960, the mouse was invented by Douglas Engelbart at the Standford Research Center.

Mouse

The computer mouse was invented by Douglas Engelbart,during the 1960s and was patented on November 17, 1970.
